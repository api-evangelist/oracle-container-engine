# Oracle Container Engine for Kubernetes (oracle-container-engine)

Oracle Container Engine for Kubernetes (OKE) is a managed Kubernetes service that simplifies the deployment, management, and scaling of containerized applications on Oracle Cloud Infrastructure with built-in security, high availability, and integration with OCI services.
